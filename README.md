# Health-Insurance-PredictionHealth Insurance Prediction
<h2>Overview</h2>

This project uses Machine Learning to predict health insurance charges based on personal information such as age, gender, BMI, smoking status, number of children, and region.

<h2>Objective</h2>

To build a predictive model that estimates medical insurance costs accurately.

<h2>Dataset</h2>

The dataset contains the following features:

 + Age
 + Gender
 + BMI
 + Number of Children
 + Smoker Status
 + Region
 + Insurance Charges (Target)

<h2>Technologies Used </h2>
 
 + Python
 + Panda
 + NumPy
 + Matplotlib
 + Seaborn
 + Scikit-learn

<h2>Machine Learning Models</h2>
Linear Regression

<h2>Evaluation Metrics</h2>

+ Mean Squared Error (MSE)
+ R² Score

<h2>Features</h2>

+ Data preprocessing
+ Exploratory Data Analysis (EDA)
+ Model training and evaluation
+ Insurance charge prediction
+ Performance comparison of multiple models

<h2>Prediction</h2>

The trained model can predict health insurance charges for **new data** entered by the user. By providing details such as age, gender, BMI, smoking status, number of children, and region, the model estimates the expected insurance charges.
